# WeatherPage - Daily Forecast Website

## 🚀 Live Demo
[View Project](https://o-marss.github.io/weatherPage/)

## 📌 About
**WeatherPage** is a weather forecast website that provides real-time weather data and dynamic background images based on location and weather conditions. It utilizes third-party APIs to enhance the user experience.

## 🛠️ Built With
- **HTML5** - For structuring the content.
- **CSS3** - For styling and layout.
- **Bootstrap** - For responsiveness and UI components.
- **Vanilla JavaScript** - For handling API requests and dynamic content.

## 🌍 APIs Used
- **WeatherAPI** - Provides real-time weather updates.
- **Unsplash API** - Fetches dynamic background images based on weather and city conditions.

## 🎨 Features
- Real-time weather data for any location.
- Dynamic background images that change based on the weather.
- User-friendly and responsive design.
- Search functionality to check weather in different cities.
